Video Bio-Tools

Objective
---------

Vidbit is a tool to convert different video formats to ilastik compatible HDF5 format.

Features
--------

* Supports the following formats: avi, ufmf, mmf, and .seq

Dependencies
------------

* OpenCV 2.4
* H5py
* Numpy
* ffmpeg
